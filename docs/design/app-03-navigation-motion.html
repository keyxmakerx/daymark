How you get around — and how it moves

Five tabs are the spine. Everything else is a drill‑in (push) or a create/edit (slide‑up sheet). Three transitions, used consistently.

The three transitions (this is “how the animations look”)

Switch tabs → fade‑through

Tapping a bottom‑nav tab cross‑fades; no directional slide.
⤳
calm · non‑spatial

Drill in → shared‑axis push

Open a detail or a tool: new screen springs in from the right, old one eases left.
→
spring · spatial forward/back

Create / edit → sheet up

Logging or writing rises as a bottom sheet over the current screen.
↑
modal · dismiss down

key idea
Tools open from two places — the More library (browse) and the “For you” router on Home (in‑the‑moment). Same screen, same push transition, so it never feels like a different app. That’s what turns the buried “More” grid into part of the daily loop.
